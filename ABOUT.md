# ABOUT NEXUS-SENTRY

## Project

**NEXUS-SENTRY** is an open-source Security & Ops Hub that centralizes alerts, metrics, and logs with AI-powered incident analysis.

## Built With

- **Backend**: Hono (Node.js) + Drizzle ORM + PostgreSQL
- **Frontend**: React + Vite + Tailwind CSS
- **AI**: Ollama (local LLM)
- **Queue**: BullMQ + Redis
- **Monorepo**: pnpm + Turborepo
